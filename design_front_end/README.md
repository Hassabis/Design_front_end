# Graduation project(webpage)

# Author：樊莲果

## 初始化项目所需要的的依赖
```
npm install
```

### 运行命令(必须先有依赖)
```
npm run serve
```

### 打包项目（打包后只需将dist文件放在服务器上即可运行项目）
```
npm run build
```

### 测试命令
```
npm run test
```

### eslint
```
npm run lint
```
